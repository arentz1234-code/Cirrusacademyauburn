# Cirrus Flight Academy Auburn

A professional flight training website for a Cirrus training center in Auburn, Alabama.

## Features

- Executive navy/gold branding
- Interactive CAPS parachute deployment animation
- Training program details (Private, Instrument, Commercial)
- Prepay pricing packages
- Student portal with Flight Schedule Pro-style booking system
- Role-based access (Admin, Instructor, Student)
- Live METAR weather display
- 24-hour calendar scheduling
- Document management system

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Montserrat, Playfair Display)

## Quick Start

```bash
# Open directly in browser
open index.html

# Or run a local server
python3 -m http.server 8000
```
